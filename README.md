## github-status-cli
> A CLI to check GitHub status.

## Installation

```sh
$ npm install -g github-status-cli
```

## Usage

```sh
$ github-status --help

  Usage
    $ github-status

    Examples
      $ github-status
      Everything operating normally.
```
